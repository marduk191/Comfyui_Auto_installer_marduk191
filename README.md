# Comfyui_Auto_installer_marduk191
A script to install everything you need for comfyui on torch 2.9.0

Includes:
-ComfyUI
-ComfyUI manager
-Pytorch 2.9.0 + vision + audio
-Flash attention
-insightface
-nunchauku
-sageattention
-numpy 1.26.4


Portable builds are avaiable here: https://huggingface.co/marduk191/Comfyui_windows_portable_builds
